# 🌿 EcoShop AI SaaS

An AI-powered branding and strategy assistant for sustainable businesses. This application helps users generate brand identities, logos, SEO strategies, and more, with full persistence via Supabase.

## 🚀 Features
- **Multi-user Authentication**: Secure sign-up/login via Supabase Auth.
- **Persistent Chat History**: Previous conversations are saved to a database and can be resumed.
- **Cloud-Stored Assets**: Branding logos are generated by AI (Amazon Bedrock) and uploaded directly to Supabase Storage.
- **Integrated Dashboard**: A central view to track and download all your generated brand assets.
- **AI Agents**: Powered by LangGraph for intelligent multi-agent orchestration.

## 🛠️ Setup Instructions

### 1. Prerequisites
- Python 3.10+
- A [Supabase](https://supabase.com/) account.
- [AWS Credentials](https://aws.amazon.com/) (for Amazon Bedrock agents).

### 2. Supabase Configuration
In your Supabase project, execute the following SQL in the **SQL Editor**:

#### Database Tables:
```sql
-- Create conversations table
CREATE TABLE conversations (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  user_id UUID REFERENCES auth.users(id),
  title TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Create messages table
CREATE TABLE messages (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  conversation_id UUID REFERENCES conversations(id) ON DELETE CASCADE,
  role TEXT, 
  content TEXT,
  extras JSONB DEFAULT '{}'::jsonb, 
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

#### Storage Bucket:
- Go to **Storage** and create a bucket named **`assets`**.
- Set its **Public** visibility to `true`.
- Add a Policy to allow **Authenticated Users** (or Public) to **INSERT** and **SELECT** objects.

### 3. Environment Setup
Create a `.env` file in the root directory and add your credentials:
```env
# AWS Credentials
AWS_ACCESS_KEY_ID=your_aws_key
AWS_SECRET_ACCESS_KEY=your_aws_secret
AWS_REGION=us-east-1

# Supabase Credentials
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your-anon-public-key
```

### 4. Installation
Install the required dependencies:
```bash
pip install -r requirements.txt
```

## 🏃 Running the App

To start the Streamlit application, run:
```bash
streamlit run app.py
```

- **Home**: Learn about the AI assistant.
- **Login/Signup**: Create your account.
- **Chatbot**: Start building your brand strategy.
- **Dashboard**: View your saved logos and assets.

## 📁 Project Structure
- `app.py`: Main Streamlit multi-page interface.
- `supabase_client.py`: Supabase Auth/DB/Storage manager.
- `graph/`: LangGraph orchestration and agent state.
- `agents/`: AI tools (Logo Generator, Strategy, SEO, etc.).
- `requirements.txt`: Python dependencies.
